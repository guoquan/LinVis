# 3D 线性代数可视化工具 (LinVis)

[English](README.md) | [中文](README_zh.md)

一个基于 Web 的交互式工具，旨在 3D 空间中可视化线性代数的基本概念。本项目使用 **React**, **Three.js** (@react-three/fiber), 和 **TypeScript** 构建。

🔗 **在线演示:** [https://guoquan.github.io/LinVis/](https://guoquan.github.io/LinVis/)

## 🎯 核心功能

*   **交互式向量操作**:
    *   通过列向量风格的 UI 动态添加、删除和修改基向量。
    *   将向量可视化为带有清晰坐标标签的 3D 箭头。
*   **张成空间 (Span) 可视化**: 自动渲染由基向量生成的子空间：
    *   **秩 (Rank) 1**: 可视化穿过原点的无限长直线。
    *   **秩 (Rank) 2**: 渲染一个倾斜的网格平面，展示两个基向量如何“编织”出平面。
    *   **秩 (Rank) 3**: 显示一个 3D 倾斜晶格，表示整个 3D 空间。
*   **线性分析**:
    *   **实时秩计算**。
    *   **线性无关性检查**: 自动检测线性相关性并提供精确的依赖方程（例如：`v3 = 2*v1 + 0.5*v2`）。
    *   **空间包含检查**: 判断“目标向量”是否位于基向量张成的空间内。
*   **投影与距离**:
    *   如果目标向量在张成空间*之外*，会显示一条**黄色投影线**（垂线），连接目标向量及其在子空间上的投影点。
*   **教室演示优化**:
    *   **投影仪友好模式**: 使用高对比度颜色（青色、橙红、品红）、粗线条和大字体标签，专为教室投影仪的可见性设计。
    *   **视角控制**: 采用 Z-轴向上 (Z-Up) 坐标系，提供一键视角复位和自动旋转功能，方便全方位展示 3D 结构。

## 🛠️ 技术栈

*   **框架**: React + Vite
*   **语言**: TypeScript
*   **3D 引擎**: Three.js / @react-three/fiber / @react-three/drei
*   **数学逻辑**: 自定义高斯消元法和线性代数工具函数

## 🚀 本地运行

1.  **克隆仓库**
    ```bash
    git clone https://github.com/guoquan/LinVis.git
    cd LinVis
    ```

2.  **安装依赖**
    ```bash
    npm install
    ```

3.  **启动开发服务器**
    ```bash
    npm run dev
    ```

4.  **构建生产版本**
    ```bash
    npm run build
    ```
